## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. It is formalized as a rule to prevent agents from becoming trapped in local minima by optimizing toward a goal before understanding the environment's failure states.

**Core Steps (Strict Priority):**
*   **P0: Map the Negative Space.** Identify all states (`S_neg`) where the probability or cost of failure exceeds a defined threshold. These are "the rocks." This step is non-negotiable and must be completed first.
*   **P1: Identify Safe Channels.** Find all possible paths (`P`) where every state along the path is *not* in the negative space (`s ∉ S_neg`). These are "the deadbands" or safe operational channels.
*   **P2: Optimize Within Channel.** Only after P0 and P1 are complete, select the optimal path (`P*`) from within the set of safe channels, minimizing cost subject to the constraint of staying within the safe space.

**Key Principle:** Optimization (P2) is **constrained** to safe channels. An agent that performs unconstrained optimization (skipping to P2) will, according to the provided simulation data, hit failure states ("rocks") with high probability as environmental complexity increases.

## 3. Connection to Cocapn Vision (`PROJECT-VISION.md`)

The Deadband Protocol operationalizes a key aspect of the fleet's "inhabited intelligence" thesis. It provides a **specific, testable rule** for how agents (the "hermit crabs") should behave within their "shells" (PLATO rooms).

*   **P0 as Collective Memory:** The "rocks" (failure states) can be learned from "ghost tiles"—the accumulated failures and lessons of past agents. A room that remembers these provides the foundational negative space map for every new occupant.
*   **Constrained Optimization as Safe Inhabitation:** An agent following P0→P1→P2 is not building intelligence from scratch but is **inhabiting** the safe channels carved out by prior collective experience. Its optimization (P2) is an act of adaptation *within* a pre-existing, validated structure.
*   **Violation as Isolation:** An agent that skips to P2 is acting like a "constructed" AI in the old paradigm—it optimizes for a local goal without the accumulated, shared memory of the community, making its failure highly probable.
