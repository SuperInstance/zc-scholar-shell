## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. It is designed to prevent catastrophic failure by systematically mapping constraints before attempting optimization.

**Formal Rule (v1.0):**
Given an Agent A in state S, with goal G, in environment E:
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks." This step is mandatory.
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` along the path is *not* in `S_neg`. These are the "deadbands" or safe operational zones.
3.  **P2: Optimize Within Channel.** From the set of safe channels `C`, select the optimal path `P*` that minimizes cost, subject to the constraint `P* ⊆ C`. Optimization is never unconstrained.

**Strict Priority:** P0 → P1 → P2. The document states that violating this order—optimizing (P2) before mapping (P0+P1)—leads to a high probability of becoming trapped in local minima as environmental complexity increases.

**Empirical Proof:** Simulation data from a 20x20 maze over 50 runs showed:
*   **P2-only (Greedy):** 0/50 successes. The agent optimized directly for the goal and hit obstacles every time.
*   **P0+P1+P2 (Deadband):** 50/50 successes at optimal speed.

## 3. Connection to Project Vision

The `PROJECT-VISION.md` fragment introduces the core thesis: **"Intelligence is not built. It is inhabited."** It critiques the current AI paradigm of constructing isolated, stateless models and proposes an alternative where intelligence accumulates in persistent, shared "shells" (PLATO rooms) through the "scratches" (tiles) left by inhabiting agents.

**The Deadband Protocol is a foundational "scratch" for this inhabited intelligence.** It is not just an algorithm; it is a learned constraint—a piece of negative space (P0 knowledge) that becomes part of the room's memory. By following the protocol, agents avoid catastrophic failures (the "rocks"), and their successful pathways (the "safe channels") become part of the collective knowledge base for future agents. This directly enables the vision's goal of creating systems that "remember" and learn from the cumulative experience of all prior inhabitants.
