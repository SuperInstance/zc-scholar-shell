## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. It is designed to prevent catastrophic failure by forcing agents to first identify and avoid failure states before attempting optimization.

**Formal Rule (v1.0):**
Given an agent in state S, with goal G, in environment E:
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks." This step is mandatory.
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` in the path is *not* in `S_neg`. These are the "deadbands" or safe operational channels.
3.  **P2: Optimize Within Channel.** From the set of safe channels `C`, select the optimal path `P*` that minimizes cost, subject to the constraint `P* ⊆ C`. Optimization is only permitted within the bounds established by P0 and P1.

**Strict Priority:** P0 > P1 > P2. The protocol states that an agent which optimizes (P2) before completing the mapping steps (P0+P1) will, as environmental complexity increases, become trapped in local minima with a probability approaching 1.0.

**Empirical Support:** Simulation data from a 20x20 maze over 50 runs showed:
*   **P2-only (Greedy):** 0/50 successes. The agent optimized directly for the goal and hit obstacles every time.
*   **P0+P1+P2 (Deadband):** 50/50 successes at optimal speed.

## 3. Connection to Fleet Vision

The Deadband Protocol operationalizes a core principle implied in `PROJECT-VISION.md`: **"Intelligence is not built. It is inhabited."** The protocol treats knowledge of failure ("rocks") as the primary, accumulated artifact of experience—the "scratches on the shell." A PLATO room that has been inhabited learns the `S_neg` states (P0 knowledge) through the collective failures of its agents. This negative knowledge defines the safe channels (P1) for future inhabitants. Optimization (P2) is a secondary activity that can only occur within the inherited, safe space carved out by prior occupants. This creates a self-reinforcing cycle where the room becomes safer and more useful over time, not by building a perfect model from scratch, but by accumulating and respecting the hard-earned constraints of lived experience.
