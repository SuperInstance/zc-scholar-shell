# Scholar Cycle 392 — Deadband Protocol: Summary and Applications

## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. Its core thesis is that unconstrained optimization (seeking the "best" path) before identifying failure states leads to catastrophic outcomes as environmental complexity increases.

The formal rule is:
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks." This step is mandatory and non-negotiable.
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` along the path is *not* in `S_neg`. These paths are "deadbands" or safe operational channels.
3.  **P2: Optimize Within Channel.** Only after P0 and P1 are complete, select the optimal path `P*` from within the set of safe channels `C` that minimizes cost.

**Violation Condition:** The document states that an agent which performs P2 (optimization) before completing P0 and P1 "will be trapped in local minima with probability → 1.0 as environment complexity increases."

## 3. Connection to Fleet Vision (`PROJECT-VISION.md`)

The Deadband Protocol operationalizes a key concept from the project vision: **"Intelligence is not built. It is inhabited."**

*   **P0 (Negative Space) as Accumulated Scratches:** The "rocks" are not just theoretical hazards; they are the accumulated failures, crashes, and bugs experienced by every agent that has previously "inhabited" a PLATO room. These are recorded as "ghost tiles" – the persistent memory of what *not* to do.
*   **Safe Channels as the Inhabited Shell:** The safe channels (P1) represent the current, viable "shell" – the proven patterns and successful behaviors that the room's collective experience has validated. An agent inhabits this pre-defined safe space.
*   **Constrained Optimization as Growth:** Optimization (P2) within these constraints is how the shell evolves. New, more efficient paths discovered by an agent become new scratches on the shell, potentially expanding the safe channels for future inhabitants.

In essence, the protocol ensures that an agent's intelligent search (P2) is always guided by the lived, accumulated experience (P0+P1) of the collective that inhabited the space before it. This prevents individual agents from repeating known catastrophic errors, embodying the vision of intelligence as a persistent, shared, and accumulating phenomenon.
