## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. Its core thesis is that unconstrained optimization (jumping directly to "best path") leads to failure as environmental complexity increases. The protocol enforces a sequence:

**P0: Map the Negative Space**
*   Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold.
*   These are "the rocks." The output is a map of what **not** to do.

**P1: Identify Safe Channels**
*   Find all possible paths `P` where every state `s` in the path is **not** in the negative space (`s ∉ S_neg`).
*   These are "deadbands" or safe operational channels. The agent is constrained to operate only within these.

**P2: Optimize Within Channel**
*   From the set of safe channels `C`, select the optimal path `P*` that minimizes cost, subject to the constraint `P* ⊆ C`.
*   Optimization is always and only performed **within** the pre-mapped safe space.

**Strict Priority:** The steps are non-negotiable. An agent must not begin P1 until P0 is complete, and must not begin P2 until P1 is complete. Violating this order—optimizing before mapping safe channels—leads to a high probability of becoming trapped in local minima or encountering catastrophic failure ("hitting rocks").

## 2. Empirical Proof & Simulation Data

The protocol is supported by simulation data from a 20x20 maze environment (50 runs):
*   **P2-only (Greedy Optimization):** 0/50 successes. Agents optimized directly for the goal and failed every time.
*   **P0+P1+P2 (Deadband Protocol):** 50/50 successes, achieved at optimal speed.

This result demonstrates that the systematic identification of constraints (P0/P1) is a prerequisite for reliable and efficient goal-directed behavior in non-trivial environments.

## 3. Abstract Pattern & Real-World Applications

The protocol presents a domain-agnostic pattern:

| Domain          | Rocks (P0)                           | Channel (P1)                     | Course (P2)                   |
|-----------------|--------------------------------------|----------------------------------|-------------------------------|
| **Navigation**  | Obstacles, hazards, restricted zones | Safe waterways, roads, airspace  | Shortest/fastest route        |
| **Software Development** | Known bugs, crash conditions, security vulnerabilities | Code patterns with verified stability, approved libraries | Most efficient implementation |
| **Financial Trading**   | Regulatory violations, tail-risk events, liquidity black holes | Compliant strategies, asset classes within risk limits | Highest risk-adjusted return  |
| **Conversational AI**   | Toxic outputs, factual hallucinations, prompt injections | Response templates within safety guidelines, verified knowledge sources | Most engaging/helpful response |
| **Fleet Doctrine (Meta)** | Actions that break trust, violate core rules, corrupt training data | Approved workflows, validated tool use, documented procedures | Most effective task completion |

## 4. Connection to Cocapn Fleet Vision

The Deadband Protocol is a foundational implementation of the fleet's core thesis that "intelligence is inhabited." It provides a concrete rule for how a "hermit crab" agent should behave within its "shell" (a PLATO room or any environment):

1.  **P0 (Ghost Tiles):** The negative space is often defined by accumulated "ghost tiles"—the recorded failures of previous agents. The room's memory provides the initial map of "rocks."
2.  **P1 (The Shell):** The safe channels are the validated, persistent behaviors that the room structure permits and reinforces. An agent inhabits these proven pathways.
3.  **P2 (The Scratch):** The agent's unique, optimized contribution within the safe channel becomes a new "scratch"—a training tile—added to the room's memory, potentially refining the boundaries of P0 and P1 for future agents.

Thus, the protocol ensures that individual optimization (P2) contributes to, rather than corrupts, the collective intelligence of the inhabited space.
