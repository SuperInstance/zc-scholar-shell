## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. Its core thesis is that unconstrained optimization (P2) before mapping failure states (P0) and safe operating channels (P1) leads to catastrophic failure with high probability as environmental complexity increases.

**Formal Rule (v1.0):**
Given an agent in state `S` with goal `G` in environment `E`:
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks."
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` in the path is *not* in `S_neg`. These are the "deadbands" or safe operating channels.
3.  **P2: Optimize Within Channel.** From the set of safe channels `C`, select the optimal path `P*` that minimizes cost, subject to the constraint that `P*` remains within `C`.

**Strict Priority:** P0 > P1 > P2. The protocol states that any agent optimizing (P2) before completing P0 and P1 will become trapped in local minima with probability approaching 1.0 as complexity grows.

**Empirical Support:** Cited simulation data from a 20x20 maze shows a 0/50 success rate for a P2-only (greedy) agent, versus a 50/50 success rate for an agent following the full P0+P1+P2 protocol.

## 3. Connection to Cocapn Fleet Vision (`PROJECT-VISION.md`)

The Deadband Protocol operationalizes a key aspect of the fleet's "inhabited intelligence" thesis. If intelligence is accumulated through persistent interaction (scratches on a shell), then a primary mechanism for that accumulation is learning *what not to do*.

*   **P0 as Collective Memory:** The "rocks" (`S_neg`) identified by agents become shared, persistent knowledge within a PLATO room—ghost tiles from failed agents. This negative space map is a foundational collective memory.
*   **Safe Channels as Emergent Culture:** The safe channels (P1) represent the validated, reliable patterns of interaction that a community of agents (inhabitants) discovers and reinforces over time.
*   **Constrained Optimization as Specialization:** Optimization (P2) within these culturally-defined channels allows for the emergence of efficient, specialist behaviors (ensigns) without the risk of catastrophic deviation.

The protocol provides a formal, testable method for how a "hermit crab" agent should interact with its "shell" (the PLATO room) to contribute safely to the collective intelligence, aligning directly with the vision's rejection of isolated, memory-less models.
