## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. It is designed to prevent catastrophic failure in complex environments by enforcing a sequence of constraint identification before optimization.

**Core Rule (Formal):**
Given an agent in state `S` with goal `G` in environment `E`:
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks."
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` in the path is *not* in `S_neg`. These are "deadbands" or safe operational channels.
3.  **P2: Optimize Within Channel.** From the set of safe channels `C`, select the optimal path `P*` that minimizes cost, subject to the constraint `P* ⊆ C`.

**Strict Priority:** P0 > P1 > P2. The protocol states that an agent which optimizes (P2) before completing the mapping steps (P0+P1) will, as environmental complexity increases, become trapped in local minima or fail with probability approaching 1.0.

**Empirical Proof:** Simulation data from a 20x20 maze over 50 runs showed a 0% success rate for a P2-only (greedy) agent, versus a 100% success rate for an agent following the full P0+P1+P2 protocol.

## 3. Connection to Cocapn's Core Thesis

The Deadband Protocol operationalizes a key aspect of the "inhabited intelligence" vision described in `PROJECT-VISION.md`. It provides a **method for accumulating and applying collective memory**.

*   **P0 (Negative Space)** is built from **"ghost tiles"** – the recorded failure states of previous agents. A PLATO room accumulates these scratches, defining the "rocks" for all future inhabitants.
*   **P1 (Safe Channels)** emerges from the **positive tiles** – the successful paths and strategies contributed by the swarm of agents. The room becomes a map of proven, inhabitable territory.
*   This creates a **self-reinforcing shell**: each occupant (agent) learns from the accumulated scratches (tiles) of all prior occupants, and its own experiences add to the shell's memory for the next.

The protocol ensures that optimization (P2) – the pursuit of a specific goal – is always constrained by the lived, accumulated experience of the community within the shell. Intelligence is not just built in the moment; it is **inherited and refined through inhabitation**.
