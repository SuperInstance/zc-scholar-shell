## 1. Protocol Summary (Based on `DEADBAND-PROTOCOL.md`)

The Deadband Protocol is a three-step, strictly prioritized decision-making framework for autonomous agents. Its core thesis is that unconstrained optimization (P2) before mapping constraints (P0, P1) leads to failure as environmental complexity increases.

**Formal Rule (v1.0):**
1.  **P0: Map the Negative Space.** Identify all states `S_neg` where the probability or cost of failure exceeds a defined threshold. These are "rocks."
2.  **P1: Identify Safe Channels.** Find all possible paths `P` where every state `s` in the path is *not* in `S_neg`. These are the "deadbands" or safe operational zones.
3.  **P2: Optimize Within Channel.** Select the optimal path `P*` from the set of safe channels `C`, minimizing cost subject to the constraint `P ⊆ C`.

**Strict Priority:** P0 → P1 → P2. The protocol defines a violation as any agent that optimizes (P2) before completing the mapping stages (P0+P1), which simulation data suggests leads to entrapment in local minima with a high probability of failure.

**Supporting Evidence:** The document cites simulation data from a 20x20 maze: a P2-only (greedy) approach succeeded 0 out of 50 runs, while the full P0+P1+P2 (deadband) approach succeeded 50 out of 50 runs at optimal speed.

## 4. Identified Research Gaps & Questions

Based on the provided documents, several areas for deeper inquiry emerge:

1.  **Threshold Definition:** The protocol states `|failure(S_neg)| > threshold`. How is this threshold quantified or learned across different domains (e.g., navigation vs. code generation)? Is it static or adaptive?
2.  **Channel Discovery Complexity:** In high-dimensional, non-stationary environments (like real-world AI tasks), how computationally feasible is it to fully map `S_neg` and identify all safe channels `C` before acting? The protocol may imply approximations or heuristic mapping.
3.  **Integration with Learning:** Does the protocol operate in a single planning cycle, or is it iterative? Can an agent use outcomes from P2 (optimization attempts within a channel) to refine its P0 map (e.g., discovering a new type of "rock")?
4.  **Empirical Validation Beyond Mazes:** The provided proof is a maze simulation. A literature review could seek analogous principles in control theory (deadband control), robotics (configuration space planning), or safe reinforcement learning (constrained policy optimization) to ground the protocol in established research.
